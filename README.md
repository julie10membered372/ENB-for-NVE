## ⚙️ Troubleshooting

If you encounter issues, check these common items first.

- Ensure you installed the NaturalVision Evolved mod before running this software.
- Check that you selected the correct installation folder for GTA V.

- Minimize background applications if you notice a drop in performance.
- Delete the ENB files from your GTA V folder to revert to the standard game visuals.

## 📈 Optimization Tips

The software includes five quality levels. If the game runs slowly, lower the quality setting.

- Low: Best for older hardware or lower-end graphics cards.
- Medium: The balance between visual quality and frame rate.
- High: Good for modern systems with strong hardware.
- Ultra: Uses the full range of reflection and light features.
- Extreme: Designed for enthusiasts with high-end displays.

Most users find that the Medium setting provides a stable experience during city driving and intensive action sequences. You can switch between these presets inside the game menu at any time.

## 📂 File Structure

The installer places several specific files into your game directory.

- enbseries.ini: Stores your primary configuration settings.
- enbseries folder: Contains the textures and shaders for the lighting affects.
- d3d11.dll: Handles the interface between the game and the light effects.
- enblocal.ini: Manages memory and hardware allocation for the software.

Do not move these files once the installer places them. The software requires this specific structure to function correctly. If you manually delete or move files, you might need to run the installer once more to repair the setup.

## 🛡️ Safety And Performance

This software modifies the way the game displays light and shadows. It does not alter game save files or interaction data. You can play your game normally while the software runs. 

The software utilizes your graphics card to process shading data. If your computer fans increase speed while you play, this indicates that the software is doing its job. Ensure your computer has adequate airflow to prevent overheating. 

If you decide you no longer want these visual changes, remove the files mentioned in the file structure section. The game will return to its original appearance immediately.
